# 鄭木清 傳記網站 — 項目說明

## 項目概覽
為外婆鄭木清建立的個人傳記網站，記錄她橫跨印尼、中國、澳門三地的一生。

## 文件結構
```
Grandma/
├── index.html          # 主網站（單頁，含所有內容）
├── Photos/             # 146 張相片
│   ├── 01.jpg – 99.jpg     # 黑白年代（印尼 & 中國）
│   ├── 501.jpg – 524.jpg   # 澳門早年（彩色）
│   ├── 801.jpg – 822.jpg   # 澳門晚年（彩色）
│   └── 1001.jpg            # 近年照片
└── Articles/
    ├── 生平.md             # 主要生平記述（原為 .pages）
    ├── 陳建回憶.md         # 長女陳建的回憶（原為 .docx）
    └── 陳燕回憶.md         # 次女陳燕的回憶（原為 .docx）
```

## 外婆資料
- **姓名**：鄭木清（聾婆婆）
- **生於**：1926年，印尼西加里曼丹省坤甸市（Pontianak）
- **祖籍**：廣東潮州揭阳县
- **丈夫**：陳坚錫（1928年生，馬來西亞，約1972年因肝癌病逝）
- **子女**：長女陳建（嫁黃燦均）、次女陳燕（嫁傅劍良）、幼子陳勇（2002年2月16日病逝）
- **孫輩**：黃偉光、黃婉華（陳建子女）；傅煒翀、傅瑞盈（陳燕子女）；陳慧彥、陳雯彥（陳勇子女）
- **遷澳**：1979年5月27日
- **2018年**：中風，入住澳門佐仔普提禪院護理中心

## 網站功能
- 溫馨復古風格（深棕 + 米白 + 金色）
- 生平時間軸
- 子女回憶片段
- 相冊按年代分四組顯示（黑白→彩色，印尼→中國→澳門早年→澳門晚年）
- 點擊相片可放大（Lightbox，鍵盤左右鍵翻頁）

## 部署至 GitHub Pages
1. 建立 public repository（例如 `grandma`）
2. 上傳整個文件夾內容（`index.html` + `Photos/`）
3. Settings → Pages → Source: `main` branch, `/ (root)` → Save
4. 網址：`https://用戶名.github.io/grandma`
