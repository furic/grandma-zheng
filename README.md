# 鄭木清 傳記網站

為外婆**鄭木清**（1926–2024）建立的個人傳記網站，記錄她橫跨印尼、中國、澳門三地的一生。

🔗 **網站**：
- https://grandma-zheng.netlify.app
- https://furic.github.io/grandma-zheng/

## 內容
- 生平時間軸
- 子女回憶片段
- 相冊（146 張相片，按年代分四組：印尼 → 中國 → 澳門早年 → 澳門晚年）
- 點擊相片可放大瀏覽

## 文件結構
- `index.html` — 主網站（單頁）
- `Photos/` — 相片
- `Articles/` — 生平與回憶記述

## 部署
透過 GitHub Pages 部署（`main` 分支 / 根目錄）。修改後推送至 `main` 即自動更新網站。
